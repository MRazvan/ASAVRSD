Software debugger v1.0

Software debugger for AVR's, it is not made as a replacement for a physical debugger, it is made for some cases where an actual debugger is not needed.
Especially for Arduinos where soldering a header and attaching a debugger is extra work that might not be possible.
Single stepping uses one interrupt so if that interrupt is needed, single step must be disabled.


Created by Razvan Marinovici - github.com/MRazvan
  • Version 1.0
    • Support only ATmega328 for now (Arduino)
    • Read Ram support
    • Write Ram support
    • Save state support
    • Single step support (uses one interrupt)
